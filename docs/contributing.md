# Contributing

See [CONTRIBUTING.md](https://github.com/celerfi/cedra-go-kit/blob/master/CONTRIBUTING.md) in the repo root for the full guide.

## Quick reference

```bash
# Clone
git clone https://github.com/celerfi/cedra-go-kit
cd cedra-go-kit

# Run tests
go test ./...

# Run tests against live testnet
CEDRA_RUN_INTEGRATION=true go test ./...
```

## Reporting vulnerabilities

Email **info@celerfi.network** — do not open a public issue.
